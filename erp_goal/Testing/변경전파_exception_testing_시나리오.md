# 변경전파 Exception Testing 시나리오 (EX-PROP)

작성: 2026-07-10. 변경전파맵(PROP-1~20)의 **비정상 경로** 테스트 — 정상 전파가 아니라
"막혀야 하는 것이 막히는지, 꼬였을 때 데이터가 깨지지 않는지"를 검사한다.
추적: 변경전파_exception_testing.json (status: passed=가드 검증됨 / guarded=코드 가드 확인·테스트 없음 / pending=미확인)

**예외 유형 분류**: ①권한 ②중복·재실행 ③동시성 ④상태 충돌 ⑤입력 오류 ⑥연쇄 중단(부분 실패)
